# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email repromptsquest@gmail.com with “gitstarter security” in the subject. We aim to respond within 48 hours.

## Supported Versions

| Version | Status |
| ------- | ------ |
| 0.1.x   | Supported |
| < 0.1.0 | End-of-life |

## Security best practices

- Never share your GitHub Personal Access Token.
- Use SSH keys for Git authentication when possible.
- Keep Git CLI up to date.
